## Resumen de la AGPL-3.0

La AGPL-3.0 es una licencia de software libre copyleft diseñada específicamente para aplicaciones web y de red.

### ✅ Lo que PUEDES hacer:

- **Usar** el software para cualquier propósito
- **Modificar** el código fuente
- **Distribuir** copias del software
- **Usar comercialmente** el software
- **Patentar** mejoras (con protección para otros usuarios)

### ⚠️ Lo que DEBES hacer:

1. **Compartir el código fuente**: Si modificas el software y lo usas en un servidor accesible públicamente (web), debes poner a disposición el código fuente de esas modificaciones bajo la misma licencia AGPL-3.0.

2. **Mantener la licencia**: Cualquier trabajo derivado debe usar la misma licencia AGPL-3.0.

3. **Incluir la licencia**: Debes incluir una copia de la licencia y el aviso de copyright.

4. **Documentar cambios**: Si modificas el software, debes documentar esos cambios.

### ❌ Lo que NO puedes hacer:

- Usar el software sin cumplir los términos anteriores
- Remover avisos de copyright
- Agregar restricciones adicionales

## Por qué AGPL-3.0

Elegimos AGPL-3.0 porque:

- Asegura que las mejoras al software se compartan con la comunidad
- Es ideal para aplicaciones web/SaaS
- Protege tanto a desarrolladores como usuarios
- Fomenta la colaboración y transparencia
